# Praha211 Web

Minimal Next.js 16 app starter for the Praha211 project.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

- `npm run dev` - start dev server
- `npm run build` - create production build
- `npm run start` - run production server
- `npm run lint` - run eslint

## Structure

- `app/layout.tsx` - root HTML layout and metadata
- `app/page.tsx` - homepage
- `app/globals.css` - global styles

## Notes

- Default create-next-app placeholder content and unused `public` SVG assets were removed to keep the base clean.
